## Text styling with HTML
```
- <br>: make a new line.
- <b></b>: bold text.
- <i></i>: italic text.
- <stron>: blod text.
- <em>: italic text.
- <small></small>: small text.
- <s></s>: strike through text.
- <sup></sup>: super script. 
- <sub></sub>: sub script.
- <del></del>: deleted text.
- <ins></ins>: inserted text.
- <mark></mark>: yellow marked text. 
```
